# Benjaba Jekyll Portfolio

A simple Jekyll portfolio starter inspired by the Carrd layout you shared.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`

## Deploy to GitHub Pages

1. Create a GitHub repo.
2. Upload these files.
3. Turn on GitHub Pages in repo settings.
4. Add your custom domain `benjaba.com`.
5. Update DNS at your domain registrar.

## Main files to edit

- `index.md` for the page content
- `assets/css/style.css` for the design
- `_config.yml` for site settings
